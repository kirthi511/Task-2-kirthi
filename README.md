# Student Grade Calculator

## Description

A Java application that calculates total marks, average percentage, and grade based on marks entered by the user for multiple subjects.

## Features

* Accepts marks for multiple subjects
* Calculates total marks
* Calculates average percentage
* Assigns grades automatically
* Displays final result clearly

## Technologies Used

* Java
* Scanner Class

## How to Run

1. Compile the program:

```bash
javac StudentGradeCalculator.java
```

2. Run the program:

```bash
java StudentGradeCalculator
```

## Concepts Used

* Variables
* Loops
* Arithmetic Operations
* Conditional Statements
